# NostrStack Minimal Site

A minimal Nostr site built with NostrStack.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## Features

- Simple Nostr feed
- Post composer
- Connect with Nostr extension (Alby, nos2x, etc.)

## Customization

Edit `src/main.tsx` to customize your site.
