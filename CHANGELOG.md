# Changelog

## 0.1.0 (2025-11-28)
- First publishable cut of `@satoshis/sdk` and `@satoshis/embed`.
- SDK: typed client for LNURLp, NIP-05, tenant/user creation; configurable host/apiKey.
- Embed: tip, pay-to-action, and Nostr comments widgets with auto-mount; clipboard copy + invoice display; Nostr login for comments.
- Backend supports multi-tenant Lightning (OpenNode), Nostr zaps, multi-name NIP-05, Prom metrics, optional OTEL.
